SmsVerifyKit Gateway — API 说明

Gateway 是 HTTP 接口，任意编程语言都能调用。密钥在服务端 .env，前端只调你自己的业务 API。

1. 检查服务

GET /health

2. 发送验证码

POST /v1/send
Content-Type: application/json

{"phone":"13800138000","scene":"login"}

scene 可选：login register bind_phone change_phone reset_password

3. 校验验证码

POST /v1/verify
Content-Type: application/json

{"phone":"13800138000","code":"1234"}

4. 若设置了 SMS_GATEWAY_API_KEY

Header: X-Api-Key: 你的密钥

返回格式

{"ok":true,"message":"验证码已发送"}
{"ok":true,"message":"验证成功","pass":true}
